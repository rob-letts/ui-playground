# Vue Starter

Key Dependencies:
- UI Framework: Vue
- Routing: Vue Router
- Store: Pinia
- Composable Library: VueUse (Core)
- Animation Library: VueUse (Motion)
- Type Safety: TypeScript
- Bundler: Vite
- Linting: ESLint
- Unit Testing: Vitest
- Component Testing: Vue Test Utils
- E2E Testing: Cypress
- Component Library: Storybook
